## Descrição

Esse é um projeto de API Rest Full, feito em NestJs.

## Intalação

```bash
$ npm install
```

## Subindo API

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
